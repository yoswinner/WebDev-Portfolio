# Yash Pradhan | Personal Portfolio

A sleek, dark-themed personal portfolio website built to showcase my academic journey, technical skills, and featured projects. This project features a modern "glassmorphism" aesthetic and custom HTML5 Canvas animations, built entirely without external frontend frameworks.

## 🚀 Features

* **Custom Canvas Animations:** Features a bespoke `GalaxyAnimation` engine rendering interactive, spiral-galaxy particle effects in the Hero and Contact sections.
* **Modern UI/UX:** Utilizes CSS variables, flexbox/grid layouts, and a dark theme with vibrant neon accents (green, cyan, purple).
* **Glassmorphism Design:** Semi-transparent, blurred card backgrounds (`backdrop-filter`) for a premium look and feel.
* **Scroll-Driven Interactions:** * Intersection Observer API used for smooth, staggered fade-in animations as elements enter the viewport.
  * Dynamic navigation bar that adapts on scroll.
  * Linear scroll progress indicator at the top of the page.
* **Fully Responsive:** Mobile-first approach ensuring the portfolio looks great on all screen sizes.
* **Zero Dependencies:** Built entirely with Vanilla JavaScript, HTML5, and pure CSS3.

## 🛠️ Tech Stack

* **Frontend:** HTML5
* **Styling:** CSS3 (Variables, Grid, Flexbox, Animations)
* **Scripting:** Vanilla JavaScript (ES6+, Canvas API, Intersection Observer)
* **Typography:** Inter (Google Fonts)
